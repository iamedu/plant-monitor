# Plant Monitor

A simple ESP32-based plant monitor that tracks soil moisture, temperature, humidity, and light.

## Initial Hardware
- ESP32 DevKit
- Capacitive soil moisture sensor
- DHT22 or SHT30
- BH1750 light sensor
- Breadboard and jumper wires
- USB cable for power/programming
